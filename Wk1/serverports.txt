1 What is the definition of a server port?
A server port is a number used by a computer to decide which service or application should receive the data.
It works like a door number on a building. Different services use different doors.
________________


2 What is the server port range?
The server port range is from 0 to 65535.
This means a computer can use 65,536 different ports.
________________


3 How many bits does the number above represent?
The port number uses 16 bits.
That is why it can have values from 0 to 65535.
________________


4 State 10 common server ports
Here are 10 common server ports:
1. 21 – FTP (File Transfer)
2. 22 – SSH (Secure remote login)
3. 23 – Telnet
4. 25 – SMTP (Send email)
5. 53 – DNS (Domain name lookup)
6. 80 – HTTP (Web pages)
7. 110 – POP3 (Receive email)
8. 143 – IMAP (Receive email)
9. 443 – HTTPS (Secure web pages)
10. 3306 – MySQL (Database)
